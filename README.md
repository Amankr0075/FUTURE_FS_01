## вљ™пёЏ Customization

### 1. Personal Information

- `Hero.jsx` вЂ” Name, title, description

- `Projects.jsx` вЂ” Add/edit projects

2. Create a free account and get your Access Key
3. In `Contact.jsx`, replace `YOUR_WEB3FORMS_ACCESS_KEY` with your key:
   ```js

   ```

### 3. Resume
Place your resume PDF at `public/resume.pdf` вЂ” the Download button will automatically work.

### 4. Colors & Theme
The entire design system is in `src/index.css` under the `:root` CSS variables. Easily change:
- `--accent-primary` вЂ” Main purple color
- `--accent-secondary` вЂ” Pink accent
- `--gradient-primary` вЂ” Primary gradient
